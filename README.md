# Student Marks Analysis

This is a simple Data Science project built using Python to analyze and visualize student marks.

## Features

* Takes student names and marks as input
* Stores data using Pandas DataFrame
* Calculates:
  * Average marks
  * Highest marks
  * Lowest marks
* Visualizes data using a bar chart

## Tech Used

* Python
* NumPy
* Pandas
* Matplotlib

## Output

* Displays student data in tabular format
* Shows statistical insights
* Generates a bar graph for visualization

## How to Run

1. Install required libraries:
   pip install numpy pandas matplotlib
   
3. Run the program:
   python filename.py
