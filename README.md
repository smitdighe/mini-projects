# Student Marks Analysis

This is a simple Data Science project built using Python to analyze and visualize student marks.

## Features

* Takes student names and marks as input
* Stores data using Pandas DataFrame
* Calculates:
  * Average marks
  * Highest marks
  * Lowest marks
* Visualizes data using a bar chart
